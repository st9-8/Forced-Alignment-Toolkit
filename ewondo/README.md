## Ewondo G2P Resources

This folder supports efforts to build an IPA grapheme-to-phoneme (G2P) model for Ewondo with Montreal Forced Aligner (MFA).

## Current Focus
- Collect lexicons and verify orthographic conventions.
- Define consistent grapheme-to-phoneme mappings in IPA.
- Package pronunciation dictionaries that MFA can ingest.
