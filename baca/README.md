## Baca G2P Resources

This folder gathers material for training an IPA grapheme-to-phoneme (G2P) model for Baca with Montreal Forced Aligner (MFA).

## Current Focus
- Source and normalize orthographic word lists.
- Convert entries to canonical IPA pronunciations.
- Prepare pronunciation dictionary files compatible with MFA.
